About Me

Hi, I'm Adcharan Arivuchelvan, a mechanical engineering student at the University of Waterloo with a passion for designing and creating innovative solutions. Using a wide range of tools, such as SolidWorks and programming, my areas of specialty are CAD, FEA, dynamics, and automation..

My Projects

EV3 Robot

Developed a Lego EV3 robot with Robot C programming that can move red cups outside the range of its ultrasonic sensor by picking them up. Using lowering mechanisms and 3D-printed claws to pick up the cups. The robot will keep performing its program until all of the cups are picked up.

Romanoake

Created a Web Karaoke app that romanizes foreign language lyrics (i.e 你好 -> Nǐ hǎo) using Spotify Api and Azure Speech Services to romanize text and play music through the Spotify app. Scoring is based on categorizing the music analysis data provided by Spotify's API.

RC Car

Created a prototype Self-Driving RC Car that uses machine learning algorithms to determine whether or not to turn or not. Uses a rack and pinion steering system with back-wheel drive. The program is run through a Raspberry Pi using a motor controller to adjust steering and a 30 FPS camera to continuously update the software.

QuakeMaster

Created a Shake Table Toy similar to Jenga Quake using a system of cam mechanisms to lift the various tiles of the Shake Table and move the blocks. Blocks were designed using interconnecting pins to enable creativity amongst users.
